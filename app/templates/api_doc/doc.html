RESTful API for easy access to user-generated reviews, café listings, and helpful rankings With Our Free API Service You can Access to all information We Collect Here About the Countries Citys Café All You Need is a Confined Account With an API key

Request

Authentication

All Requests Must Contain header with your string type API key

Headers Example :

headers = {"api-key" : "Your API Key"}

API

Errors

Your request can return anm error if your request don't have headers with your API key or Your API key was not valid or one or more parameters in your request not valid

Errors List :

Error 400 - Bad Request : Your Request Headers Dont have your "api-key" check the key and Try Again !
Error 401 - Unauthorized : Your API key is invalid Check Your Key and Try Again !
Error 404 - Not Found : Your request data was not found in the server
Error 406 - Not Acceptable : You Miss Some Required Prams  In Your request check api Documentation and Try Again !

API End Points :

Description :

Return countries information in our database

End Point :

https://favoritecafe.ir/api/v1.0/country/all

Parameters :

prams = {"count" : (optional default is 10 type : integer) }

Response Parameters : 

'country-name': name of the country,
'capital': capital of the country,
'phone-cod': phone of a country,
'iso-2': iso-2 standard country cod,
'about': information about country

Request Example :

Example without count :

curl --location 'https://favoritecafe.ir/api/v1.0/country/all' --header 'api-key: your api key'

Example with count (Number of Results Will be 20) : 

curl --location 'https://favoritecafe.ir/api/v1.0/country/all?count=20' --header 'api-key: your api key'

Description :

Return Single countries information in our database with given iso-2 standard cod

End Point :

https://favoritecafe.ir/api/v1.0/country/single

Parameters : 

prams ={
       "iso_cod": ('Required'country iso-2 cod for Example Canada iso-2 cod is 'CA' type : string)
       }

Response Parameters : 

'country-name': name of the country,
'capital': capital of the country,
'phone-cod': phone of a country,
'iso-2': iso-2 standard country cod,
'about': information about country

Request Example :

curl --location "https://favoritecafe.ir/api/v1.0/country/single?iso_cod='CA'" --header 'api-key: your api key'

Description :

Return Many city inside country information in our database with given iso-2 standard cod

End Point :

https://favoritecafe.ir/api/v1.0/city/all

Parameters : 

prams ={
       "count" : ('optional' default is 10 type : integer),
       "iso_cod": ('Required'country iso-2 cod for Example Canada iso-2 cod is 'CA' type : string)
       }

Response Parameters : 

'id': id of the city you need this if you want to search for a single city,
'city-name': name of the city,
'population': population of the city,
'latitude': latitude of the city,
'longitude': longitude of the city,
'about': information about the city,
'total-cafe': number of cafe inside the city,

Request Example :

Example without count :

curl --location "https://favoritecafe.ir/api/v1.0/city/all?iso_cod=CA" --header 'api-key: your api key'

Example with count (Number of Results Will be 20) : 

curl --location 'https://favoritecafe.ir/api/v1.0/city/all?iso_cod=CA&count=20' --header 'api-key: your api key'

Description :

Return all cafe information inside a city in our database with given city id

(You can find city id by sending request to get all city inside a country end point)

End Point :

https://favoritecafe.ir/api/v1.0/cafe/all

Parameters : 

prams ={
       "id": ('Required'id of the city type : string)
       }

Response Parameters : 

'id': id of the cafe you need this if you want to search for a single cafe,
'create_by': creator of the cafe can be user oe owner
'name': name of the cafe,
'overall rating': overall rating for cafe from 100 its collecting from users votes
'cafe_theme': cafe_theme of the cafe,
'open_time': opening time  of the cafe,
'close_time': closing time of the cafe,
'address': address of the cafe,
'phone': phone number of the cafe,
'coffee_score': coffee score of the cafe,
'wifi_score': wifi score of the cafe,
'power_score': power score of the cafe,
'total_vote': number of total vote users casting of this cafe,
'can_take_calls': a boolean can you answer your phone calls inside this cafe ,
'about': information about this cafe,
'followers': number of followers of the cafe,
'reviews': reviews of this cafe,
'cafe_menu': a list with all this cafe menu items and prices in USD$

Request Example :

curl --location "https://favoritecafe.ir/api/v1.0/cafe/all?id=7795f682-4b01-4fdf-9ce0-bdd1f0dd1520" --header 'api-key: your api key'
